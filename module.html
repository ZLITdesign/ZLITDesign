穿梭选择

双栏穿梭选择，用直观的方式在两栏中移动元素，完成选择行为。

1.方向键选择：选择一个或以上的选项后，点击对应的方向键，可以把选中的选项移动到另一栏。

2.勾选选择：左栏勾选完成选择，右栏可取消选择；右栏取消选择后左栏同步状态；以常用的选择人员为例。

3.选择项目较多的情况下可增加搜索功能。

//config的设置是全局的
layui.config({
	base: 'layui/lay/mymodules/' //假设这是你存放拓展模块的根目录
}).use(['transferTool','form'], function () {
	var transferTool = layui.transferTool;
	var form = layui.form;

	transferTool.init({
		elem: "#tranferDiv",
		seletedValue: "1,2",  //已选,变量逗号隔开
		url: 'data/list.json',  //url获取数据
		name: 'province',  //form表单获取的结果值
		// disabledValue: "2,4"   //不可选,变量逗号隔开
	});

	//监听提交///保存或修改
	form.on('submit(submitForm)', function (data) {
		// layer.alert(JSON.stringify(data.field));
		// console.log(data.field);

		var arr = $(data.form).serializeArray();
		var province = [];
		arr.forEach(function (val,i) {
			province[i] = val.value;
		});
		layer.alert(JSON.stringify(province));
		return false;
	});

	//获取选中得值
	$(document).on("click", "#getSelValue", function (e) {
		e.preventDefault();
		alert(transferTool.getSelectedDataId());      //获取选中值的ID
		alert(transferTool.getSelectedDataTitle());   //获取选中值得文字值
	});

	//获取未选中得值
	$(document).on("click", "#getUnSelValue", function (e) {
		e.preventDefault();
		alert(transferTool.getUnSelectedDataId());    //获取选中值的ID
		alert(transferTool.getUnSelectedDataTitle()); //获取选中值得文字值
	});
});

列表/表格 >

列表/表格

表格/列表是展现数据最为清晰、高效的形式之一。它常和排序、搜索、筛选、分页等其他界面元素一起协同，适用于信息的收集和展示、数据的分析和归纳整理、以及操作结构化数据。它结构简单，分隔归纳明确，使信息之间更易于对比，大大提升了用户对信息的接收效率和理解程度。

1.表格中的时间、状态、操作需保持词语完整不过行。

2.当数据为空时，可使用『- -』来表示暂无数据。

3.可搭配批量操作，多选框勾选后点击操作按钮完成批量操作；勾选后提示勾选数量和其他辅助信息；可清空选择。

4.对齐方式：字段长度固定居中对齐，长度不确定左对齐，带统计意义数字右对齐。

5.列表项数量较多无法显示完整时，可提供显示设置；用户自定义设置显示部分列表项，查看列表详情显示完整信息。
